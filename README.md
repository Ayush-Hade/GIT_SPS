🎮 Stone Paper Scissor Game
A simple and interactive Stone Paper Scissor (Rock Paper Scissors) game built using web technologies. Users can play against the computer and track their scores in real time.

🌐 Live Demo
👉 https://git-sps.vercel.app/

💡 About the Project
This project is a fun web-based implementation of the classic Stone Paper Scissor game, where the user competes against the computer. The application provides instant results and keeps track of scores dynamically.

✨ Features

🎯 Play against computer
⚡ Instant result (Win / Lose / Draw)
📊 Live score tracking
🧠 Random computer choices
💻 Clean and simple UI


🛠️ Tech Stack

HTML
CSS
JavaScript
Deployment: Vercel


🚀 How It Works

User selects one option: Stone / Paper / Scissor
Computer randomly selects its move
Game logic compares both choices
Result is displayed instantly
Scores are updated in real-time


🧠 Game Logic

Stone beats Scissor
Scissor beats Paper
Paper beats Stone
